# Система за управление на кораби (IARA System)

Това е уеб-базирана система за управление на регистъра на корабите, разработена с Flask. 
Системата позволява добавяне на нови кораби в базата данни и изтриване на съществуващи записи.

## Функционалности
- **Добавяне:** Въвеждане на име на кораба, регистрационен номер и име на капитан.
- **Преглед:** Списък с всички регистрирани кораби в удобна таблица.
- **Изтриване:** Възможност за премахване на кораби от регистъра.

## Технологии
- **Backend:** Python, Flask, Flask-SQLAlchemy
- **Frontend:** HTML5, Bootstrap 5
- **База данни:** SQLite
