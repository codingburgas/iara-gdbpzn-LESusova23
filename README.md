# IARA Information System

Централизирана платформа за дигитализиране и управление на процесите в Изпълнителна агенция по рибарство и аквакултури (ИАРА).

## Технологичен стек
- Backend: Python, Flask
- База данни: SQLite, SQLAlchemy
- Frontend: Bootstrap 5, CSS3 (Glassmorphism)

## Основни модули
- Риболовен флот: Регистър на корабите и технически характеристики.
- Разрешителни: Издаване и проследяване на лицензи.
- Електронен дневник: Отчитане на улова по кораби.
- Инспекции: Регистър на проверките и административни актове.
- Любителски риболов: Издаване на билети за любители.
- Аналитичен панел: Статистика в реално време.
